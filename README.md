# Learn Python

Bienvenido a este repositorio de práctica en Python. Aquí consolidé ejercicios, ejemplos y pequeños proyectos mientras estudio el libro "Automatiza cosas aburridas con Python" y exploro conceptos fundamentales del lenguaje.

Este proyecto está pensado como un espacio de aprendizaje progresivo: desde los primeros pasos hasta la creación de scripts más completos y proyectos interactivos.

## ✨ ¿Qué encontrarás aquí?

- Ejercicios básicos de sintaxis y variables
- Control de flujo con condicionales
- Uso de bucles y estructuras repetitivas
- Funciones reutilizables y modularización
- Técnicas de depuración y resolución de errores
- Manipulación de listas y estructuras de datos
- Pequeños proyectos prácticos para aplicar lo aprendido

- [automatiza-cosas-aburridas/01-basic-concepts](automatiza-cosas-aburridas/01-basic-concepts) - Conceptos básicos de Python

- [automatiza-cosas-aburridas/projects](automatiza-cosas-aburridas/projects) - Proyectos pequeños y aplicaciones sencillas

## 🚀 Cómo usar este repositorio

1. Clona el repositorio en tu máquina.
2. Abre los archivos de ejercicios en tu editor de código.
3. Ejecuta los scripts con Python:

```bash
python3 nombre_del_archivo.py
```

4. Modifica los ejemplos, experimenta y trata de resolverlos por tu cuenta.

## 🧠 Objetivo de aprendizaje

El enfoque principal es fortalecer las bases de programación en Python mediante práctica constante y proyectos pequeños, con un enfoque claro en:

- Pensamiento computacional
- Resolución de problemas
- Escritura de código legible
- Automatización de tareas repetitivas
